# Mandarin Learning Flashcards

This repository contains a simple flashcard web app for studying Mandarin vocabulary from the Taiwan CCCC (Children's Chinese Competency Certification) word list.

It was created as a personal study tool to review the vocabulary in a cleaner, mobile-friendly format.

Source data for this project comes from the TOCFL teaching resources download page:
https://tocfl.edu.tw/tocfl/index.php/teach/download

The app is based on the CCCC vocabulary resource listed there under `兒童華語文能力測驗情境詞彙表`.
